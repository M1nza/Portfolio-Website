AutoWeave

Autoweave is an AI-driven platform that enables the generation of virtual car designs tailored to user preferences. It combines aesthetic styling with functional insights by factoring in materials and cost. The system is structured into three modules—User, Workshop, and Worker—to ensure smooth interaction and streamlined output delivery. Leveraging Hugging Face’s robust models for natural language understanding, Autoweave transforms user inputs into optimized car visualizations, output as high-quality images—without the use of AR or heavy machine learning pipelines.

Data Cleaning and Visualization

In this project, I analyzed customer data to understand bike purchasing behavior using Microsoft Excel. I primarily used Excel Solver to build a simple optimization model, helping to determine the best conditions or customer segments for maximizing bike sales. This involved setting constraints and objectives based on the dataset. The project enhanced my ability to apply Excel for real-world decision-making scenarios without using advanced coding tools.
